Iterator object
- added since ES6
- special kind of object which implements the iterable protocol
- this iterator object implements next() method
    the next() returns an object with 2 properties ---> value, done
- A lot of JS built-in objects have iterator property
    Strings
    Arrays
    Maps
    Sets
    NodeLists
    argument Object
